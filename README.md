# ads-ai — HLD-страница

AI-оператора рекламных кампаний Google Ads на Rust: approve-гейт, дуальная

с жёстким потолком в коде.

Живая страница: https://yunusovt983-art.github.io/ai-google-ads/

Один файл `index.html`, без сборки и внешних зависимостей.
